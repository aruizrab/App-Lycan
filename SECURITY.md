## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability, please do NOT create a public GitHub issue.

Instead, please report it via email to [EMAIL_PLACEHOLDER] ensuring you include:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.

We will acknowledge receipt of your report within 48 hours and strive to provide a fix as soon as possible.
